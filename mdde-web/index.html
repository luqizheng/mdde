📖 使用说明

步骤 1: 下载构建脚本

根据您的操作系统下载对应的环境构建脚本

步骤 2: 克隆项目源码

git clone https://github.com/your-org/oa.git cd oa

步骤 3: 运行构建脚本

# Windows .\env-build.ps1 # Linux/macOS chmod +x env-build.sh ./env-build.sh

步骤 4: 按提示配置环境

脚本会自动检测系统并生成相应的配置文件

步骤 5: 开始开发

# 使用生成的脚本 .\run-cmd.ps1 dotnet restore .\run-cmd.ps1 dotnet build .\run-cmd.ps1 dotnet run
